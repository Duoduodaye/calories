# 🍎 Calories - 热量追踪小程序

> 微信小程序 - 简洁优雅的热量管理工具

一个功能完整的热量追踪微信小程序，支持食物摄入和运动消耗的全方位记录分析。

## ✨ 功能特性

### 🍽️ 全面餐次记录
- **6种餐次**：早餐、午餐、晚餐 + 上午/下午/夜宵加餐
- **智能分类**：主餐与加餐视觉区分
- **一键记录**：点击餐次卡片快速添加

### 🏃‍♂️ 运动消耗追踪  
- **23+种运动**：有氧、球类、力量训练、休闲、日常活动
- **个性化计算**：基于用户体重调整消耗热量
- **直观显示**：emoji图标 + 分钟热量消耗

### 📊 热量平衡分析
- **实时计算**：摄入 - 消耗 = 热量差
- **周统计**：平均摄入/消耗/热量差分析  
- **目标追踪**：每日热量目标达成率

### 🎨 莫兰迪设计风格
- **温和配色**：莫兰迪绿主题 (#9CAF88)
- **性别标识**：♂♀符号简洁标记
- **响应式布局**：优雅卡片设计

## 🚀 技术架构

```
calories/
├── app.js              # 小程序入口文件
├── app.json            # 全局配置 (莫兰迪绿主题)
├── app.wxss            # 全局样式
├── utils/
│   └── data.js         # 数据管理 + 食物/运动数据库
├── pages/
│   ├── index/          # 首页 - 热量概览 + 运动总结
│   ├── record/         # 记录页 - 食物/运动双模式
│   ├── stats/          # 统计页 - 周数据 + 热量平衡

└── images/             # 底部导航图标
```

## 📱 快速体验

### 开发环境
1. 安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 导入项目 → 测试AppID → 开始体验
3. Web演示：打开 `demo/index.html`

### 核心功能
- ✅ **食物记录**：20+食物数据库，营养成分完整
- ✅ **运动追踪**：体重调整的热量消耗计算
- ✅ **热量平衡**：每日摄入-消耗差值分析
- ✅ **数据统计**：7天趋势 + 平均值计算

## 🛠️ 开发特色

- **原生框架**：微信小程序原生开发，性能优秀
- **本地存储**：数据保存在设备，隐私安全
- **模块化**：清晰的代码结构，易于扩展
- **莫兰迪设计**：温和优雅的视觉体验

## 📈 数据模型

**食物数据** → 营养计算 → 餐次记录  
**运动数据** → 消耗计算 → 平衡分析  
**统计数据** → 趋势展示 → 目标追踪
